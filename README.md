# Raspberry Pi Lab Code Pack

This pack contains code files for the experiments we discussed, excluding the LCD section as requested.

## Included
- LED blink
- Button-controlled LED
- Button-controlled LED brightness using PWM
- 7-segment + buzzer volume control
- Ultrasonic people counter with SQLite
- MQ2 gas sensor
- DHT11 readout
- PIR sensor + camera + LED
- RFID RC522 read
- MQ3 alcohol sensor
- TCS3200 color sensor
- BMP280 pressure sensor
- R307 fingerprint sensor

## Notes
- Some scripts need extra libraries installed on the Raspberry Pi.
- Pin numbers are written in BCM numbering.
- Check your lab wiring before running any script.
